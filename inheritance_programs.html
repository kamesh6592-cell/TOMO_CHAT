Here are 3 very simple Java programs that show inheritance.

1) Single inheritance (basic)

class Animal {
    void eat() {
        System.out.println("Animal is eating");
    }
}

class Dog extends Animal {    // Dog inherits from Animal
    void bark() {
        System.out.println("Dog is barking");
    }
}

public class Test1 {
    public static void main(String[] args) {
        Dog d = new Dog();
        d.eat();    // inherited method
        d.bark();   // own method
    }
}

2) Multilevel inheritance

class Animal {
    void eat() {
        System.out.println("Animal is eating");
    }
}

class Dog extends Animal {    // Dog inherits Animal
    void bark() {
        System.out.println("Dog is barking");
    }
}

class BabyDog extends Dog {    // BabyDog inherits Dog
    void weep() {
        System.out.println("BabyDog is weeping");
    }
}
public class Test2 {
    public static void main(String[] args) {
        BabyDog bd = new BabyDog();
        bd.eat();    // from Animal
        bd.bark();   // from Dog
        bd.weep();   // from BabyDog
    }
}

3) Method overriding with inheritance

class Animal {
    void sound() {
        System.out.println("Some generic animal sound");
    }
}

class Cat extends Animal {    // Cat inherits Animal
    @Override
    void sound() {            // overriding method
        System.out.println("Cat meows");
    }
}

public class Test3 {
    public static void main(String[] args) {
        Animal a1 = new Animal();
        Animal a2 = new Cat();   // upcasting
        
        a1.sound();   // calls Animal's sound
        a2.sound();   // calls Cat's sound (overridden)
    }
}
